# Smart Ticket Management System

## Project Description
Smart Ticket Management System is a web-based application designed to create, manage, track, and resolve support tickets efficiently. It helps users report issues, monitor ticket status, and improve communication between users and administrators while ensuring organized and effective issue management.

## Features
- User Registration and Login
- Create Support Tickets
- Track Ticket Status
- Update Ticket Information
- Resolve and Close Tickets
- User-Friendly Interface

## Technologies Used
- Python
- MySQL
- HTML
- CSS
- JavaScript

## Project Objective
The main objective of this project is to simplify ticket management and provide an organized platform for handling user requests and support issues.
